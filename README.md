# Portfólio Coletivo

Este projeto é uma página web simples que reúne os portfólios dos integrantes do grupo, facilitando o acesso aos projetos e perfis profissionais de cada membro.

## Descrição do Projeto

O site apresenta uma página principal com links para os portfólios individuais dos integrantes e seus perfis no GitHub.  
O objetivo é centralizar e organizar os trabalhos desenvolvidos por cada membro do grupo.
